### 2. 快速修复方法

**sed 添加代码：**
```bash
sed -i '/^}$/i\    // 新方法' file.java
```

**路径确认：**
```bash
grep -n "@GetMapping" *Controller.java
```

**git 恢复：**
```bash
git checkout <file>
```

**成果：** 快速修复 19 个接口
